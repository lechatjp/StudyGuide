Study Guide

The Internet

The internet is an interconnected network of computer networks that spans the globe. Today it is a major part of our lives.

Network

A network consists of two or more computers connected for the purpose of communicating and sharing resources.

LAN

A local area network (LAN) is a computer network that interconnects computers within a limited area such as a residence, school, laboratory, university campus or office building.

Client

The client, or user, side of the Web. It typically refers to the Web browser in the user's machine.

Server

A web server is server software, or hardware dedicated to running said software, that can satisfy World Wide Web client requests. A web server can, in general, contain one or more websites.

IETF

The Internet Engineering Task Force (IETF) is the principal body engaged in the development of new Internet protocol standard specifications.

W3C

The World Wide Web Consortium (W3C)Takes a proactive role in developing recommendations and prototype technologies related to the Web.

RFC

A Request for Comments (RFC) is a formal document from the Internet Engineering Task Force ( IETF ) that is the result of committee drafting and subsequent review by interested parties.

Accessibility

Web accessibility is the inclusive practice of ensuring there are no barriers that prevent interaction with, or access to, websites on the World Wide Web by people with physical disabilities, situational disabilities, and socio-economic restrictions on bandwidth and speed.

Protocols

Protocols are rules that describe how clients and servers communicate with each other over a network.

HTTP

Hypertext Transfer Protocol

FTP

File Transfer Protocol

IP

Internet Protocol

TCP

Transmission Control Protocol

SMTP

Simple Mail Transfer Protocol

POP

Post Office Protocol

IMAP

Internet Message Access Protocol

TCP/IP

TCP/IP is the official internet protocol suite. TCP, IP and the many other protocols of TCP/IP have specific functions and work together to ensure reliable network communication.

MIME

Multipurpose Internet Mail Extensions (MIME) is a standard that helps to classify the types of files used on the internet.

PACKETS

A packet is the unit of data that is routed between an origin and a destination on the Internet or any other packet-switched network
